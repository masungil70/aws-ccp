# AWS CCP - 비용 관리 및 지원 서비스

## 비용 분석 도구 (AWS Pricing Calculator, Cost Explorer, AWS Budgets, CUR)

AWS CCP 시험에서는 **"AWS에서 비용을 어떻게 예측하고, 분석하고, 통제하는가?"**가 매우 중요한 학습 포인트입니다.

AWS는 단순히 사용한 만큼 비용을 청구하는 것이 아니라, **예상 → 분석 → 모니터링 → 상세 분석**의 전체 비용 관리 체계를 제공합니다.

전체 흐름은 다음과 같습니다.

```
서비스를 만들기 전
      │
      ▼
AWS Pricing Calculator
(예상 비용 계산)
      │
      ▼
서비스 운영 시작
      │
      ▼
Cost Explorer
(실제 비용 분석)
      │
      ▼
AWS Budgets
(예산 초과 알림)
      │
      ▼
CUR
(가장 상세한 비용 데이터 분석)
```

---

# 1. 비용 분석 도구 개요

AWS 비용 관리 도구는 크게 4가지입니다.

| 도구                   | 목적                  |
| ---------------------- | --------------------- |
| AWS Pricing Calculator | 예상 비용 계산        |
| Cost Explorer          | 실제 비용 분석        |
| AWS Budgets            | 예산 관리 및 알림     |
| CUR                    | 상세 비용 데이터 제공 |

쉽게 말하면

> **Calculator → 얼마 나올까?**
>
> **Cost Explorer → 얼마 나왔지?**
>
> **Budgets → 너무 많이 쓰면 알려줘**
>
> **CUR → 모든 비용 데이터를 CSV 형태로 분석**

입니다.

---

# 2. AWS Pricing Calculator

## 개념

AWS 서비스를 구축하기 전에

**예상 비용을 계산하는 서비스**

입니다.

즉,

> "EC2 3대를 한 달 동안 사용하면 얼마인가?"

를 계산하는 도구입니다.

---

## 언제 사용하는가?

예를 들어 회사에서

```
웹 서버

EC2
↓

RDS

↓

S3
```

구성하려고 합니다.

아직 만들지도 않았는데

관리자가 묻습니다.

> "한 달에 얼마 나오나요?"

이때 사용하는 것이

**Pricing Calculator**

입니다.

---

## 예시

다음과 같은 구성을 입력합니다.

```
EC2 t3.medium
2대

RDS db.t3.medium
1대

S3
500GB
```

↓

예상 결과

```
EC2
$60

RDS
$45

S3
$12

총 비용

$117 / 월
```

이처럼

미리 견적을 계산합니다.

---

## 특징

* 실제 비용이 아님
* 예상 비용 계산
* 아키텍처 설계 시 사용
* 고객 제안서 작성
* 예산 계획 수립

---

## CCP 시험 포인트

시험에서는 자주 묻습니다.

> "서비스를 구축하기 전에 예상 비용을 계산하는 서비스는?"

정답

> **AWS Pricing Calculator**

---

# 3. Cost Explorer

## 개념

실제로 사용한 비용을

그래프로 분석하는 서비스입니다.

즉

이미 AWS를 사용 중인 계정의 비용을 보여줍니다.

---

## 확인 가능한 내용

예를 들어

```
이번 달 비용

EC2
$320

S3
$40

RDS
$110
```

또는

```
지난 6개월

비용 증가 추이
```

도 확인 가능합니다.

---

## 분석 기준

Cost Explorer에서는

다양한 기준으로 분석할 수 있습니다.

예를 들어

### 서비스별

```
EC2

S3

Lambda

RDS
```

---

### Region별

```
서울

도쿄

버지니아
```

---

### 태그(Tag)

예를 들어

```
Project=A

Project=B
```

별로도 비용을 분석할 수 있습니다.

---

## 특징

* 실제 사용 비용
* 그래프 제공
* 필터 제공
* 추세 분석
* 비용 예측(Forecast)

---

## CCP 시험 포인트

문제

> "AWS 사용 비용을 시각적으로 분석하는 서비스는?"

정답

> **Cost Explorer**

---

# 4. AWS Budgets

## 개념

예산을 미리 설정하고

초과하면 알림을 보내는 서비스입니다.

예를 들어

```
이번 달

예산

$100
```

설정합니다.

현재 비용

```
$85
```

↓

아무 일도 없습니다.

그러나

```
$101
```

이 되면

이메일을 보냅니다.

```
Budget exceeded!

현재 사용량

101%
```

---

## 알림 조건

다양하게 설정 가능합니다.

예를 들어

```
50%

80%

100%

120%
```

도 가능합니다.

---

## 특징

* 이메일 알림
* SNS 알림
* 예산 관리
* 실제 비용 기반
* 예측 비용 기반

---

## 사용 사례

회사에서

매달

```
AWS 비용

100만원
```

이 넘으면

관리자에게 메일을 보냅니다.

---

## CCP 시험 포인트

문제

> "예산을 초과하면 알림을 제공하는 서비스는?"

정답

> **AWS Budgets**

---

# 5. CUR (Cost and Usage Report)

## 개념

CUR은

AWS에서 제공하는

가장 상세한 비용 데이터입니다.

CSV 형태의 리포트를 제공합니다.

---

## 포함되는 정보

예를 들어

```
EC2

사용 시간

인스턴스 종류

Region

계정

태그

비용

할인

Savings Plan 적용 여부

Reserved 적용 여부
```

모든 데이터가 기록됩니다.

---

## 어디에 저장되는가?

CUR은

보통

```
S3
```

버킷에 저장됩니다.

이후

다음과 같이 분석합니다.

```
CUR

↓

S3

↓

Athena

↓

QuickSight
```

즉,

SQL로 분석이 가능합니다.

---

## 사용 목적

기업에서는

다음과 같은 질문을 분석합니다.

```
부서별 비용

프로젝트별 비용

태그별 비용

계정별 비용

서비스별 비용
```

---

## 특징

* 가장 상세한 데이터
* CSV/Parquet 제공
* S3 저장
* Athena 분석 가능
* QuickSight 시각화 가능

---

## CCP 시험 포인트

문제

> "AWS 비용 데이터를 가장 상세하게 제공하는 서비스는?"

정답

> **Cost and Usage Report(CUR)**

---

# CCP 시험 암기 포인트

다음 네 가지를 연결해서 기억하면 문제를 빠르게 풀 수 있습니다.

| 질문                                                | 정답                        |
| --------------------------------------------------- | --------------------------- |
| **얼마가 들지 미리 계산하고 싶다**                  | AWS Pricing Calculator      |
| **실제로 얼마를 사용했는지 그래프로 보고 싶다**     | Cost Explorer               |
| **예산을 넘으면 자동으로 알려주고 싶다**            | AWS Budgets                 |
| **가장 상세한 사용 내역을 SQL이나 BI 도구로 분석하고 싶다** | CUR (Cost and Usage Report) |

### 한 줄 암기법

> **"계산(Calculator) → 분석(Explorer) → 알림(Budgets) → 상세 데이터(CUR)"**

이 흐름만 기억해도 AWS CCP 시험에서 비용 관리 관련 문제의 대부분을 해결할 수 있습니다.
