# AWS 클라우드의 장점

안녕하세요. 오늘은 AWS CCP 시험에서 매우 중요하게 다루는 **AWS 클라우드의 장점(Benefits of AWS Cloud)** 에 대해 알아보겠습니다.

대학생 여러분이 학교에서 프로젝트를 진행한다고 생각하면서 설명해 보겠습니다.

---

# 1. 초기 투자 비용이 거의 없다 (비용 절감)

기존 방식에서는 서버를 직접 구매해야 합니다.

예를 들어,

> AI 프로젝트를 위해 서버 1대를 구매한다면

| 항목      | 비용    |
| ------- | ----- |
| 서버 장비   | 500만원 |
| 스토리지    | 100만원 |
| 네트워크 장비 | 100만원 |
| 유지보수    | 매년 발생 |

총 700만원 이상이 필요할 수 있습니다.

하지만 AWS에서는 서버를 구매하지 않습니다.

필요한 만큼 빌려서 사용합니다.

### 예시

* 시험 기간 : 서버 10대 사용
* 방학 기간 : 서버 1대 사용
* 프로젝트 종료 : 서버 삭제

사용한 만큼만 비용을 지불합니다.

이를 **Pay-as-you-go(종량제 요금제)** 라고 합니다.

CCP 시험 핵심 키워드

> **CapEx → OpEx 전환**

| 구분    | 의미       |
| ----- | -------- |
| CapEx | 초기 자본 지출 |
| OpEx  | 운영 비용 지출 |

---

# 2. 몇 분 만에 서버 생성 가능

기존 환경에서는 서버를 설치하는데 시간이 오래 걸립니다.

### 기존 방식

```text
서버 주문
↓
배송 대기
↓
랙 설치
↓
운영체제 설치
↓
네트워크 구성
↓
서비스 배포

약 2주~2개월 소요
```

### AWS

```text
AWS Console 접속

↓

EC2 생성

↓

5분 이내 사용 가능
```

---

### 대학생 예시

캡스톤 프로젝트 발표 전날

학생:

> "내일 시연해야 하는데 서버가 부족해!"

AWS:

```text
EC2 생성
↓

3분 후 서버 준비 완료
```

---

# 3. 필요할 때 자동 확장 가능 (탄력성)

클라우드의 가장 큰 특징 중 하나입니다.

이를 **Elasticity(탄력성)** 라고 합니다.

---

### 상황 예시

학생 커뮤니티 웹사이트 운영

평소 사용자

```text
100명 접속
```

시험기간

```text
10,000명 접속
```

---

기존 서버

```text
서버 다운 발생
```

AWS

```text
Auto Scaling

↓

서버 자동 증가

↓

트래픽 감소

↓

서버 자동 축소
```

---

장점

* 성능 유지
* 비용 절감
* 관리 편의성 향상

---

CCP 시험 키워드

> Elasticity = 자동 확장/축소

---

# 4. 전 세계 어디서나 서비스 가능

AWS는 전 세계 여러 지역에 데이터센터를 운영하고 있습니다.

이를 **리전(Region)** 이라고 합니다.

---

예를 들어,

한국 학생들이 만든 게임 서비스를 운영한다고 가정해 봅시다.

사용자 지역

* 한국
* 미국
* 일본
* 유럽

AWS에서는 가까운 리전을 선택하여 서비스를 배포할 수 있습니다.

예시

* 서울 리전
* 도쿄 리전
* 미국 버지니아 리전
* 프랑크푸르트 리전

장점

* 응답 속도 향상
* 글로벌 서비스 제공 가능

---

# 5. 높은 가용성 제공

서비스는 항상 실행되어야 합니다.

AWS는 여러 개의 데이터센터를 운영합니다.

이를 **가용 영역(AZ, Availability Zone)** 이라고 합니다.

---

예시

서울 리전

```text
AZ-A

AZ-B

AZ-C
```

웹 서버 구성

```text
AZ-A  서버1

AZ-B  서버2
```

만약 AZ-A 장애 발생

```text
AZ-B 서버가 서비스 지속
```

사용자는 장애를 거의 느끼지 못합니다.

---

CCP 핵심 개념

| 용어                | 의미        |
| ----------------- | --------- |
| Region            | 지리적 영역    |
| AZ                | 독립된 데이터센터 |
| High Availability | 높은 가용성    |

---

# 6. 보안 수준 향상

AWS는 다양한 보안 서비스를 제공합니다.

대표 서비스

| 서비스       | 역할         |
| --------- | ---------- |
| IAM       | 사용자 권한 관리  |
| KMS       | 암호화 키 관리   |
| Shield    | DDoS 공격 방어 |
| WAF       | 웹 공격 차단    |
| GuardDuty | 위협 탐지      |

---

예시

학생 프로젝트에 참여하는 인원이 5명이라고 가정합니다.

AWS IAM을 이용하면

```text
학생 A → 관리자

학생 B → 개발자

학생 C → 읽기 전용
```

처럼 권한을 세밀하게 관리할 수 있습니다.

---

# 7. 다양한 AWS 서비스를 즉시 활용 가능

AWS는 200개 이상의 서비스를 제공합니다.

대표적인 서비스

| 분야     | 서비스        |
| ------ | ---------- |
| 가상 서버  | EC2        |
| 저장소    | S3         |
| 데이터베이스 | RDS        |
| 서버리스   | Lambda     |
| 컨테이너   | ECS, EKS   |
| AI/ML  | SageMaker  |
| 모니터링   | CloudWatch |

---

### 대학생 예시

캡스톤 프로젝트에서

* 웹 서버 → EC2
* 파일 저장 → S3
* 데이터베이스 → RDS
* AI 모델 학습 → SageMaker

를 조합하여 실제 기업 수준의 서비스를 구축할 수 있습니다.

## AWS CCP 시험에서 꼭 기억해야 하는 문장

> **AWS 클라우드는 필요한 만큼 사용하고, 사용한 만큼 비용을 지불하며, 전 세계에 안전하고 확장 가능한 서비스를 빠르게 제공할 수 있는 플랫폼이다.**

이 문장만 이해해도 AWS CCP 시험의 클라우드 가치 제안(Value Proposition) 관련 문제를 상당수 해결할 수 있습니다.
